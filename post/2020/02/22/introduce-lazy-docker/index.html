LazyDocker, 도커를 위한 CLI 대시보드 알아보기

2020년 02월 22일 Harry The Great

도커로 로컬에서 개발을 하다보면 도커를 모니터링하기 다소 불편합니다. docker-compose나 stack으로 개발을 하다보면 전체 도커의 모니터링하기가 불편합니다. 물론 Portainer같은 GUI툴이 있으면 더 편하겠지만 터미널에서 간편하게 사용할 수 있는 lazydocker가 있습니다.

# 컨테이너정보

왼쪽에서 컨테이너를 선택하면 크게 로그, 스탯, 설정정보, 도커 내에서 동작하는 프로세스를 확인할 수 있습니다. 단축키로 [ 와 ] 키를 이용하면 쉽게 메뉴를 변경할 수 있습니다.

상세메뉴

컨테이너를 선택한 상태에서 b키를 누르면 bulk커맨드창으로 docker-compose와 관련한 메뉴를 실행할 수 있습니다. docker exec메뉴와 마찬가지로 명령어를 실행할 수 있습니다. b는 Bulk 커맨드로 만약 일반커맨드키인 c키를 누르면 docker exec -it [컨테이너] bash를 바로 호출할 수 있습니다.

추가적으로 알아두면 좋은 단축키
  • d - 컨테이너 삭제
  • r - 컨테이너 재시작
  • s - 컨테이너 정지
  • R - 재시작옵션보기(build 다시하기와 같은 옵션)

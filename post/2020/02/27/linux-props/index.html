Unix시스템의 Glob (Filename Extension)

2020년 02월 27일 Harry The Great

# 패턴

모든걸 선택하는 *

rm *.txt

txt확장자를 가지는 모든파일 삭제하기

한글자만 선택하는 ?

rm *.?
#ello.c => 삭제
#hello.php => 삭제되지않음

한글자로 끝나는 확장자

디렉토리까지 포함하여 모든걸 선택하는 **

rm model/**.c

model 폴더 안에 위치한 모든 파일중 c확장자를 가지는 파일을 삭제합니다.

# 응용하기

foo로 시작하는 모든 파일

foo*

foo로 시작하고 txt를 확장자로 갖는 파일

foo*.txt

foo로 끝나는 모든파일

*foo

a/b/d는 포함이 되지만 a/b/c/d는 포함X

a/*/d

a/b/d/ 또는 /a/b/c/d 모두 포함

a/**/d

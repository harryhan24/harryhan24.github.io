Swift throwable function 예시

2020년 02월 16일 Harry The Great

# 구현예제

enum RequestError: Error {
    case urlError
}

//반환 타입에 throws를 포함해줍니다.
func getRequest(stringUrl: String) throws -> URLSession.DataTaskPublisher {

    //URL 타입이 아닐 경우 에러를 내보냅니다.
    guard let url = URL(string: stringUrl) else {
        throw RequestError.urlError
    }


    let request:URLRequest = URLRequest(url: url)
    return URLSession.shared.dataTaskPublisher(for: request)
}

throwable functions은 try, try?, try! 3개 중 하나로 받아야 합니다.

  • try - 일반적인 try catch 처리
  • try? - 실패시 null 리턴
  • try! - 실패시 에러발생

# 사용예제

let publsiher = try? getRequest(stringUrl: "https://myapi.test.com")

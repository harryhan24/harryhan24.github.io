FD 커맨드 사용예제

2020년 02월 14일 Harry The Great

placeholder 이용하여 확장자 png로 바꾸어주기

fd -e jpg -x convert {} {.}.png

place

fd -e zip -x unzip

모든 flac파일을 opus 파일로 변환하기

fd -e flac -x ffmpeg -i {} -c:a libopus {.}.opus

wp-content 내의 내용을 제외하고 검색하기

fd -e php index -E wp-content

모든파일 권한변경

fd -e jpg -x chmod 644 {}

Placeholder

fd -e jpg -x chmod 644 {}와 같은 커맨드를하고 파일이 01.jpg밖에 없을경우 {} 안에는 wp-content/uploads/01.jpg와 같은 파일 상대주소가 들어오게됩니다.

Placeholder 사용예시
  • {.} – 확장자뺀값을 받습니다.
  • {/} - path를 제외한 값을 받습니다.
  • {//}: 파일 이름뺸 디렉토리만 (wp-content/uploads).
  • {/.} Only the basename, without the extension (01).

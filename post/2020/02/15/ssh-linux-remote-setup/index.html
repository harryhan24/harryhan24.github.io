원격 리눅스서버 키파일 생성 및 패스워드 로그인차단

2020년 02월 15일 Harry The Great

# 키파일생성 및 원격연결

ssh-keygen -t rsa

저장할 이름을 입력한 후 빈칸 혹은 원하는 암호화 텍스트를 입력합니다. 

ssh-copy-id -i ~/.ssh/[키파일명].pub root@[아이피주소]

원격에 패스워드 자동으로 연결 만약 ssh-copy-id를 입력하지 않는다면 원격호스트의 .ssh 폴더에 직접 옮긴 후 권한수정합니다.

$HOME/.ssh/config 파일 하단에 아래와같이 추가합니다.

Host [원격서버 별칭]
HostName [원격서버 IP주소]
User root
IdentityFile ~/.ssh/[키파일이름] //공개(*.pub)키 아닌 비밀키

# 비밀번호 로그인 차단

/etc/ssh/sshd_config 파일을 아래와같이 수정합니다. 이미 설정이 되어있는 값도 있기때문에 되도록 하단 추가보다 vi의 search를 이용합니다.

PermitRootLogin without-password
ChallengeResponseAuthentication no
PasswordAuthentication no

service sshd restart

마지막으로 ssh 데몬을 재시작합니다. 이후 별칭이 아닌 ::ssh 아이피주소:: 형식으로 로그인을 시도할때 Permission denied, please try again. 메세지가 나온다면 성공

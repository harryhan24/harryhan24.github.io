라즈베라파이 설정 및 쿠버네티스 k3s 설치하기

2020년 01월 14일 Harry The Great

# SD카드 포맷 및 이미지 준비

SD카드를 연결한 후 Mac이라면 Disk Utility -> Erase -> Format으로 이동 후 포맷타입을 꼭 MS-DOS한 후 Erase합니다. 윈도우라면 SD Card Formatter를 설치합니다. 다음으로 SD카드에 이미지를 세팅해줄Etcher  프로그램과 [Raspbian][2]을 다운받습니다. 라즈비안(Raspbian)버전은 CLI 기반으로 사용한다면 Raspbian Buster Lite를 내려받습니다.

Rasbian 이외에도 리눅스에서 제공하는 라즈베리파이용 우분투버전이 존재합니다.

Etcher 프로그램을 실행한 후 Select Image를 클릭한 후 다운받은 이미지를 선택합니다. 끝날때까지 기다린다음 SD카드를 라즈베리파이에 넣어줍니다. 만약 모니터없이 세팅을 하고싶다면 소문자로 ssh라는 파일을 SD카드 루트에 생성합니다. 초기 아이디와 패스워드는 pi/raspberry입니다.

ssh 설정이 제대로 안되는경우: sd카드가 정상적으로 연결되었는지 확인합니다. 이후 모니터를 연결 후 sudo raspi-config -> advanced option >혹은 interface option에 접근하여 ssh enable로 설정합니다.

# 로그인 후 간단한 설정하기

원격접속을 한 후 raspi-config명령어로 configuration 메뉴로 진입합니다. 만약 CLI로 사용한다면 GUI를 위한 메모리가 필요없으므로 advance -> memory split -> 16mb 로 설정합니다. Network Option메뉴에서는 네트워크상 이름을 변경할 수 있습니다. 비밀번호없이 접속하고싶다면 ssh-keygen -R 아이피주소커맨드를 이용하여 키를 만듭니다.

# 쿠버네티스 설치하기

k3s는 경량화된 버전의 쿠버네티스입니다. 싱글노드로도 사용할 수 있고 별도의 네트워크 설정으로 노드연결도 가능합니다. 쿠버네티스에서 직접 시스템을 변경할 수 있도록 /boot/cmdline.txt파일에 아래와같은 라인을 추가합니다.

cgroup_enable=cpuset cgroup_memory=1 cgroup_enable=memory
